# login_OAth
This API Authanticates users on login.
It creates and returns a web token to user after authentication user,
then creates a middleware to evaluate web token


Tech-stack:
  1. NodeJS
  2. MySQL 
  
Dependancies:
  1. npm i nodemon
  2. npm i bcrypt
  3. npm i express
  4. npm i jsonwebtoken
  5. npm i mysql
